# 02. D Flip-Flop

## Overview

This project implements a basic D flip-flop in Verilog.

A D flip-flop is one of the most fundamental sequential logic elements in digital design. It stores one bit of data and updates its output on a clock edge.


## Design Goal

The goal of this project is to understand how clocked sequential logic is described in RTL.

The flip-flop captures the input `d` on the active clock edge and stores it in the output register `q`.
