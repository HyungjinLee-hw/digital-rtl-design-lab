\# Digital RTL Design Lab



This repository contains my Verilog RTL design practice projects, starting from basic FPGA bring-up and sequential logic, then progressing toward CPU building blocks.



This repository is part of my preparation for digital RTL design, computer architecture, SoC front-end design, and eventually AI/HPC accelerator design.



\## Current Focus:

A step-by-step CPU design project based on individually designed and verified RTL building blocks.

\- Parameterized ALU



Planned:

\- Instruction memory

\- Control unit

\- Datapath integration
